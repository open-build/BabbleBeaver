System Prompt Configuration

Edit the initial system prompt that defines BabbleBeaver's behavior and personality. This prompt is loaded from initial-prompt.txt.

Auto mode (default): If frontend sends a "context" field, it will automatically be used. Accepts ANY JSON object - no specific fields required.

Context-Aware Feature

Enabled by default (auto mode). When frontends send a context field with ANY JSON data, it will be automatically formatted and included in the system prompt.

Example: Frontend sends {"context": {"page": "settings", "user_role": "admin"}} and BabbleBeaver automatically knows the user is on the settings page.

Smart behavior: If no context is provided, works like normal. If context is provided, automatically uses it.

Note: Context mode changes require server restart to take effect.
